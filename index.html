Shortcuts.IO - Video action on IMDB.com

Triggering the shortcuts mouse click requires that a user click and hold down (long press) on the mouse until the shortcut box appears. By doing this the app avoids conflict with the traditional browser click. I developed the technique from experiences with long press on touch devices.

Shortcuts.IO - Example mini app built with code editor.

Mini apps run like normal built in actions, except rather then being stored in the main code base(like the search,video and image actions). They get stored in the browsers local storage area (in the case of chrome I used the SQLite DB).
